vision for Airta Platform

Test Service Platform (TSP)

With the Airta Intelligent Analytics Platform (AIPA), software testing is deployed into the cloud which covers functional, performance, and security testing areas. In the meantime, it will be integrated with the software development team providing Continuous Integration to constant improve the DevOps operation and maintenance processes. For specific enterprise customer, the AIPA can provide the provision of customer acceptance automation and on-demand integration testing.

Automated Testing and Analysis (ATA)

The Airta platform system will provide comprehensive code analysis data, including UI Crawler, code Analyzer, and logic detector to provide logical process analysis, data/code input/output analysis, and user action process analysis to maximize the use of artificial intelligence to simulate the real user process, complete the test automation coverage and defect tracking.
